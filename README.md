# Javascript Quiz (ES6)

*A collection of javascript base quiz*

## Array
  - **Q**: How to copy an array?
    ```javascript
    const items = [1,2,3];
    const itemsCopy = [...items];
    ```

  - **Q**: How to convert an array-like object to an array? (e.g. arguments in function)
    ```javascript
    func() {
        return Array.from(arguments);
    }
    ```
